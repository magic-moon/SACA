# 高等教育图书电子化
这类图书是高校专业书籍，学校里每年都有很多同学去复印老旧书籍，这样粗制的盗版，既影响阅读体验，也不利于知识的传播，我想利用自己的业余时间为这本书用LaTeX重编一遍，公布源码和PDF文本供同学们使用。
